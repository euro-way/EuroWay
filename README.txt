# EuroWay (ready-to-deploy)

This folder contains a single-page static site for **EuroWay**.

How to use:
1. Unzip `EuroWay_final.zip`
2. Push the contents into a GitHub repository (e.g. `gh-pages` branch) or any static hosting.
3. The files are:
   - index.html
   - style.css
   - script.js
   - images/ (logo and other assets)
   - reviews.json (you can replace with your real reviews from GitHub)

Notes:
- The "Залишити заявку" button opens Viber:
  `viber://chat?number=%2B380689234960`
- The "Замовити дзвінок" button uses `tel:+380689234960`
- To use reviews from your GitHub repo, replace `reviews.json` with your file from repo.
